# 项目知识库

此目录存放项目沉淀的知识文档，主要供 AI Agent 检索使用。

## ⚠️ 注意

**请勿手动添加或修改此目录下的文档。**

如需新增或更新知识文档，请通过 AI 对话让 Agent 帮忙处理，以触发 `knowledge-manager` skill 完成：

1. 文档内容的创建/更新
2. 下方知识索引的同步更新

这样可以确保知识库索引与文档内容保持一致。
