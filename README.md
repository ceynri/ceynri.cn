### Technology

- Web Framework: [Astro](https://astro.build/)
- Styling: [TailwindCSS](https://tailwindcss.com/) + [SASS](https://sass-lang.com/)
- Comments Engine: [giscus](https://github.com/laymonage/giscus)
- AI tools: [Cursor](https://www.cursor.com/)
